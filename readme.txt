描述：创建一个后台管理程序，只需要简单的几步。 

使用方法：

1. 上传aqadmin目录到你的程序根目录
2. 将aqadmin.sql传到数据库
3. 设置aqadmin根目录的aq.config.php
4. 默认登录账号，密码  admin  admin
